# 会计与税务：商业思考的起点而非终点

## 原文出处

> "会计数字是商业语言，而且对于评估企业价值和追踪其发展，有着莫大的帮助。如果没有这些数字，芒格和我会迷路的：对于我们而言，它们是我们评估自己公司和其他公司的起点。但是，公司经理人和投资者应该记住，会计仅仅是有助于商业思考，但却永远不能替代这种思考。"（第6章E节）

> "当一家公司被骗子领导时，盈利数字可能就像抹灰腻子一样任意捏造。当然，真相最终会浮出水面，但与此同时，大笔金钱或许已被骗走。"（第6章C节）

> "用'笔'盗窃一大笔钱，远远比用'枪'抢劫一小笔钱，安全得多。"（第7章导言）

> "我们同意投资作家雷·德沃的观察：'人们过度追求收益所造成的损失，超过了被人持枪打劫！'"（第1章G节）

## 核心概念

### 1. 经济商誉 vs 会计商誉

**会计商誉**：收购价超过净资产公允价值的差额，GAAP要求在40年内摊销（"减值"）

**经济商誉**：企业以极少有形资产产生超额回报的能力——来源包括消费特许权、低成本地位、政府特许权

> "当企业被预期可以产生的回报，远远高于市场回报率时，这样的公司在逻辑上具有远高于其有形资产的价值。这种超额回报的资本化就是经济商誉。"

禧诗糖果是最好的例子：
- 1972年：800万有形资产 → 200万利润（ROE 25%）
- 1995年：500万有形资产 → 5000万利润（ROE 1000%）
- 同期会计商誉从3600万摊销到2300万，但经济商誉暴涨

### 2. 股东盈余（Owner Earnings）

巴菲特发明的最重要的非GAAP指标：

```
股东盈余 = (a)财报盈利 + (b)折旧摊销等非现金费用 - (c)维持竞争力所需的年均资本化开支
```

关键点：
- (c) 是必须的猜测——"一定是猜测，而猜测有时是非常困难的"
- 如果 (c) > (b)：GAAP盈利夸大了真实经济盈利（石油业是典型反面教材）
- 如果 (c) ≈ (b)：GAAP接近真实（如禧诗糖果）
- "现金流"（a+b）忽略 (c) = 荒谬——"华尔街报告中经常出现的'现金流'数字非常荒谬"

> "尽管牙医总是说，如果你不关心你的牙齿，它们都会掉光的，同样的情况也适用于(c)。"

### 3. 透视盈余（Look-through Earnings）

```
透视盈余 = 运营盈利 + 持股公司归属的留存利润 - 假设分红的税款
```

解决了持股20%以下但盈利远大于分红时GAAP严重低估的问题。

### 4. EBITDA 是危险信号

> "我们不认为EBITDA是一种有意义的业绩评价方法。"

原因：把折旧当作非真实成本——但它"是一项真实的经济成本，就像工资、原材料或税金一样"。那些强调EBITDA的经理人容易做出错误决策。

### 5. 期权估值：布莱克-斯科尔斯的缺陷

巴菲特的批判：
- 历史波动率用于短期期权"有用但非万无一失"
- 用于长期期权"会对较大幅度的错误估值"——100年的标普500看跌期权不可能只值250万美元
- 因为：①通胀使名义价值上升②留存利润使企业价值大幅增长③100年后指数下跌概率远小于1%
- "宁愿要模糊的正确，也不要精确的错误"

## 三问检查法

巴菲特认为财务报告应帮助读者回答三个问题：
1. 这家公司大约值多少钱？
2. 它实现未来规划的可能性有多大？
3. 鉴于现有的情况，公司管理层的工作表现有多好？

> "在大多数情况下，从最低要求的GAAP报告中，找到上述一个或更多问题的答案，其难度介乎困难与不可能之间。"

## 审计委员会的四问

（第1章G节，巴菲特提出的审计委员会应询问审计师的四个问题）

1. 如果审计师是唯一负责编制报表的人，做法会与现在不同吗？
2. 如果审计师是投资者，是否收到过重要信息帮助理解当期财务表现？
3. 如果审计师是CEO，公司是否遵循了应有的内部审计流程？
4. 审计师是否观察到任何将收入/成本从一个报告期转移到另一个报告期的行为？

> "这些四个问题的主要目的是起到预防作用。一旦审计师们知道，审计委员会要求他们为公司管理层的行为背书，而不是默默姑息时，他们会在这个过程一开始就拒绝做坏事。"

## 美国钢铁公司讽刺案例

巴菲特在1990年年报附录中虚构了美国钢铁公司"现代化会计方案"，讽刺地展示如何通过会计诡计"创造"利润：
- 固定资产减值到**负**10亿美元 → 折旧变"增值"
- 存货记为1美元 → 售出即产生"利润"
- 用零息债券50%折扣赎回 → 制造"回购利润"
- 建立10亿美元"应急储备金"吸收一切

这则讽刺点出了核心问题：**管理层将GAAP视为需要克服的障碍，而非需要达到的标准。**

## 边界条件

- **会计不是真理**：GAAP只是起点，不是终点
- **折旧 ≠ 商誉摊销**：折旧是真实成本；摊销可以忽略
- **ROE优于EPS增长**：测试管理层水平的真正标准
- **高ROE需要排除杠杆和会计花招**
- **"模糊的正确胜过精确的错误"**贯穿始终

## 双链关联

- ← 是 [内在价值]计算的语言工具——但永远只是"起点"
- ← 关联 [管理层评估]——诚信的管理层不玩会计把戏；不诚信的"厨房里不止一只蟑螂"
- → 影响 [资本配置]——透视盈余是判断资本配置效率的依据
- → 与 [安全边际]对应：会计数字越不可靠 → 需要越大安全边际

## L2 认知痕迹

巴菲特对会计的态度展现了一种典型的**二阶思维**：

大多数人：看到财报数字 → 认为这就是事实
巴菲特：看到财报数字 → 问"这个数字在多大程度上反映了经济现实？"→ 问"谁编制了这个数字，他们有什么动机？"

他对EBITDA的批判特别能体现这种思维——"强调现金流的经理人容易做出错误决策"不是关于会计的陈述，而是关于**人性**的陈述：当你用不合理的标准衡量自己时，你会做出不合理的决策。

"宁愿要模糊的正确，也不要精确的错误"在会计领域的应用逻辑：GAAP提供了精确的错误；巴菲特要的是模糊的正确。这不是反会计，而是反**对会计数字的盲目崇拜**。
