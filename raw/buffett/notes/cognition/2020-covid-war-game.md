# 2020年COVID危机：为什么没有重复2008？

> 批次9用框架推演了2008年危机，推演与实际高度一致。
> 但同样的框架在2020年却**没有**产生类似的"大规模买入"决策。
> 为什么？框架哪里需要修正？

---

## 框架推演（2020年2-3月）

### 时间线

```
2020.02.19: 标普500历史高点 3,386
2020.02.24: 巴菲特CNBC专访——"疫情不应该改变你的投资方式"
2020.03.09: 标普500第一次熔断（-7%）
2020.03.11: WHO宣布COVID为全球大流行
2020.03.12: 第二次熔断
2020.03.15: 美联储紧急降息至0% + $7000亿QE
2020.03.16: 第三次熔断（-12%）
2020.03.23: 标普500触及低点 2,237（从高点下跌34%）
2020.03.23: 美联储宣布无限QE
2020.05.02: 伯克希尔年会——巴菲特宣布已清仓所有航空股
```

### 框架推演：2008逻辑怎么应用于2020？

| 2008年 | 2020年 | 框架判断 |
|--------|--------|---------|
| 金融危机→信贷冻结 | 实体封锁→营收消失 | **不同**——2008是金融体系的偿付危机，2020是实体经济的收入休克 |
| 价格跌到"明显低于"内在价值 | 跌了34%，但内在价值也变了——未来现金流因封锁而高度不确定 | **无法判断**——DCF三问中"灌木丛里确定有鸟吗？"的答案不再是确定的 |
| 政府介入（TARP）创造安全网 | 政府介入更快更深（3月15日降息+QE，3月23日无限QE） | **窗口更短**——2008花了三周才通过救助方案；2020美联储三天出手 |
| 巴菲特有大量现金（$400B+） | 巴菲特有$137B现金 | 弹药充足但**出手犹豫** |
| 买了高盛/GE优先股 | 卖掉了航空公司，回购了伯克希尔股票 | **完全不同的行为** |

## 实际发生了什么

### 巴菲特的真实行动（2020年）

1. **2月24日CNBC专访**："疫情不应该改变你的投资方式""如果你买一个企业，你买的是它未来10年、20年的盈利。"
2. **3月**：市场三次熔断，历史性波动——巴菲特**没有公开的买入**。
3. **4月**：巴菲特出售了约$6.5B的航空股（达美、西南、美联航、美国航空）——全部清仓。
4. **5月2日年会**：公开确认清仓航空股。"我们卖掉了全部航空股。我们买的时候觉得这是一个好行业，但世界变了。"
5. **Q2-Q3**：大量回购伯克希尔股票（$5.1B in Q2 alone）。

### 为什么没有重复2008模式？

**核心判断**：2008是金融体系的偿付危机——只要有流动性和时间，资产价值会恢复。2020是实体经济的收入休克——即使有流动性和时间，有些企业的"未来现金流"已经永久性改变了。

航空公司的例子最能说明问题：巴菲特在股东会上说："我们买航空股的时候是$70-80，我们卖的时候也是一个大致相当的价位。"但他仍然**全部清仓**——不是因为价格跌了，而是因为"世界变了"。航空旅行的需求可能需要数年才能恢复，而航空公司的固定成本（飞机租赁、人工）不会消失。

这触及了框架的深层问题：

```
能力圈     → ✅ 航空业在能力圈内（巴菲特投资过航空公司）
护城河     → ⚠️ 航空公司的护城河在COVID下受到根本性质疑
管理层     → ✅ 管理团队没有变
价格       → ✅ 价格跌了34%，但...

内在价值?  → ❌ 无法确定！封锁的持续时间完全未知——
            如果封锁6个月，航空公司的内在价值是多少？
            如果封锁2年，航空公司还能存活吗？
            DCF的"确定性"假设被打破了。
```

## L3发现：框架的隐藏前提被暴露了

### 前提1：未来是可预测的（至少"模糊地"可预测）

巴菲特的整套DCF框架建立在"灌木丛里确定有鸟"这个前提上。2008年危机虽然可怕，但它是一个**金融现象**——只要金融体系不崩溃，实体经济的运行逻辑不变。2020年是一个**生物/政治现象**——政府可以任意决定封锁的持续时间；病毒可以变异；疫苗可能成功也可能失败。这一切都不在"企业分析"的范畴内。

### 前提2：恐慌 = 机会

"在别人恐惧时贪婪"在2008年完美成立。但在2020年，恐慌的源头不是市场情绪，而是**客观存在的、不可预测的物理风险**。市场先生不是"抑郁"——他是**合理地不确定**。

### 前提3：现金能在恐慌中买到好东西

2008年，你有现金 → 可以买高盛的优先股，因为高盛别无选择 → 你能谈出极好的条件。2020年，美联储在**三天内**投放了数万亿美元的流动性。企业不需要去找巴菲特借钱——美联储就是免费的提款机。**政府介入的速度决定了私人资本的议价能力。**

## L3 种子

**"在系统性恐慌中，政府介入的速度决定了巴菲特的框架是否适用。如果政府在恐慌窗口内提供了无限流动性，'在别人恐惧时贪婪'的机会窗口可能被关闭。"**

**研究问题**：
1. 如果美联储在2008年9月17日（雷曼后2天）就启动无限QE，巴菲特还会在9月23日投资高盛吗？
2. 2020年的经验是否意味着"大而不倒"时代的巴菲特危机策略（2008模式）已经终结，取而代之的是"回购自己+等待"的新模式（2020模式）？
3. 如果下一次危机是地缘政治性质（如台海冲突），框架的哪个前提会被打破？

## 对框架的修正建议

| 原规则 | 问题 | 修正 |
|--------|------|------|
| "在别人恐惧时贪婪" | 恐惧的源头是物理风险而非情绪时，贪婪可能是错的 | 增加前置判断：恐惧的源头是可恢复的金融恐慌，还是不可逆的实体破坏？ |
| 四关决策流程 | 第四关"价格"假设内在价值是可估算的 | 增加一个前置检查：内在价值本身是否因为外部冲击发生了**结构性而非周期性的变化**？ |
| 现金 = 期权 | 政府介入速度可能使现金期权的价值大幅贬值 | 区分"缓慢移动的危机"（2008，政府行动慢）和"快速响应危机"（2020，Fed三天出手） |
