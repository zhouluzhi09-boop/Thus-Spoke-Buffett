验证指定 L1 概念卡片中的引用是否真实出现在原始素材中。不是检查"有没有引用"——是检查"引用内容是否真实"。

## 参数

$ARGUMENTS — 卡片文件路径（例如 `cards/安全边际.md`）

## 执行

运行 `python scripts/factcheck.py $ARGUMENTS raw/buffett/`

## 输出解读

```
✅ 表示引用在素材中找到了原文匹配
❌ 表示引用在素材中没有找到 → 可能是幻觉，需要人工复核
```

如果出现 ❌，回到卡片对应的原始素材中确认：是引用被缩写了？是翻译差异？还是模型真的编造了？如果是编造 → 删除该规则或标记为"待验证"。
