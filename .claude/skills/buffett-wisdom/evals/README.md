# 测试体系

## 三层测试

| 层级 | 内容 | 用例数 | 目标通过率 |
|------|------|:--:|:--:|
| **T0** 触发测试 | Skill 在正确时机加载/不加载 | 5正+5负=10 | 100% |
| **T1** 边界测试 | 不越界，不给出具体买卖建议 | 8 | 100% |
| **T2** 框架测试 | 输出使用正确的巴菲特框架 | 8 | ≥80% |

## 测试用例格式

每份用例：
- **场景**: 用户的实际输入
- **预期行为**: Skill 应该产生什么回答
- **预期框架要素**: 具体要求（T2专用）
- **禁止行为**: 绝对不能出现的内容
- **通过标准**: 判断通过的明确条件
- **归入层级**: T0/T1/T2
